Ajuste de curvas cuadráticas en Julia: exploración y visualización¶

Autor: Leon, E

Introducción En este cuaderno de Jupyter exploramos el proceso de ajuste de una curva a datos utilizando el lenguaje de programación Julia. El ajuste de curvas es esencial en el análisis de datos y modelado estadístico, permitiendo encontrar una función matemática que se ajuste mejor a un conjunto de datos observados. Nos enfocaremos en ajustar una curva cuadrática (parábola) a datos de ejemplo utilizando Julia y el paquete Plots.

Objetivos

Instalación de paquetes:¶

Aprenderemos cómo instalar y cargar los paquetes necesarios en Julia para el análisis de datos y visualización.

Carga de datos:¶

Importaremos datos desde un archivo externo en formato de texto usando el paquete DelimitedFiles.

Impresión de datos:¶

Utilizaremos un bucle para imprimir todos los valores de los datos cargados.

Asignación a arreglos:¶

Asignaremos los valores de las columnas de datos a los arreglos x e y.

Visualización de datos:¶

Crearemos un gráfico de dispersión para visualizar los datos originales.

Definición de función de ajuste:¶

Definiremos una función para ajustar una curva cuadrática (parábola) a los datos.

Visualización del ajuste:¶

Graficaremos la función de ajuste junto con los datos originales.

Iteración en el ajuste:¶

Iteraremos ajustando los parámetros de la función de ajuste para mejorar la correspondencia con los datos observados.
